The Story Behind

It all started in 2010 when I started working on aero images and intensively used my coding skills to improve productivity. During that period, I created massive amouts of customized tools for GIS and photogrammetry. Since there were different platforms to work on, I came up with my own way to master a new programming langue as quickly as possilbe.

However, as I dug deeper into this field, I found it less attractive and decided to do something more challenging. Therefore, I went back to school and chosed computer science as my major. Meanwhile, I also worked as a web developer and was doing some game design just for fun. I had strong passion for software development that time, but then a friend of mine introduced me to to an IC design company, and I started to work as a firmware engineer.

While software developers are writing code for humans, firmware people write programs for machines. They are both writing code to achieve certain goals but in different ways. To some extent, I like firmware better because you know, it's based on more basic build blocks. Generally, the more basic it is the more challenging it can be. Sometimes, I wonder if this is a gene in my blood, I learn difficult things just because I like challenges.
